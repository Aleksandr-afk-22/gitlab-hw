## Задание 1

Что нужно сделать:

1. Зарегистрируйте аккаунт на [GitHub](https://github.com/).
2. Создайте **новый отдельный публичный репозиторий**. Обязательно поставьте галочку в поле «Initialize this repository with a README».
3. Склонируйте репозиторий, используя https протокол `git clone ...`.
4. Перейдите в каталог с клоном репозитория.
5. Произведите первоначальную настройку Git:  
   `git config --global user.name "Имя"` и  
   `git config --global user.email email@example.com`.
6. Выполните `git status` и запомните результат.
7. Отредактируйте `README.md`, переведя его в состояние *Modified*.
8. Повторите `git status` после каждого шага.
9. Просмотрите изменения: `git diff` и `git diff --staged`.
10. Добавьте файл в индекс: `git add README.md`.
11. Снова проверьте `git diff` и `git diff --staged`.
12. Сделайте коммит: `git commit -m 'First commit'`.
13. Отправьте изменения: `git push origin master`.
