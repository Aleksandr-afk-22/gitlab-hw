**Решение:**

### Конфигурация CI/CD Pipeline для Go приложения

**Этапы выполнения:**
```yaml
stages:
  - test    # Запуск автоматических тестов
  - build   # Сборка приложения
```

**Задача тестирования:**
```yaml
test:
  stage: test
  image: golang:1.17
  script:
    - go test .
```

**Задача сборки:**
```yaml
build:
  stage: build
  image: golang:1.17
  script:
    - go build .
```
